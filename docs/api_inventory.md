# API Inventory

GET /dashboard/summary

* Returns revenue, customers, orders

GET /customers/segments

* Returns customer segment distribution

GET /customers/churn

* Returns churn statistics

GET /customers/clv

* Returns CLV tier distribution

GET /products/top

* Returns top products

GET /products/recommendations

* Returns recommendation rules

GET /products/pairs

* Returns product affinity insights

GET /forecast/revenue

* Returns historical and forecasted revenue
